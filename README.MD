# diskominfo-bantul-submission

# Intisari

Submission dari Achmad Kurnianto untuk Tenaga Ahli Diskominfo Bantul 2026 Tahap 2.

Project ini menggunakan bahasa pemrograman TypeScript.
Adapun library/framework yang digunakan antara lain:
1. React.js
2. Next.js
3. Material UI
4. drizzle ORM

## Prosedur Menjalankan Aplikasi

### Persyaratan
1. Pastikan install nodejs LTS terbaru.
2. Pastikan terhubung dengan database yang disediakan.
3. Salin file `.env.example` ke `.env`, kemudian beri nilai `DATABASE_URL`-nya.
4. install library yang digunakan menggunakan `npm i --force` atau `pnpm install`

### Run untuk development

```bash
    pnpm dev
```

### Run for production.

```bash
    pnpm build
```
